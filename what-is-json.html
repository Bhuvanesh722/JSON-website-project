What is JSON?

JSON (JavaScript Object Notation) is a lightweight format for storing and transporting data. Ideally, it's easy for humans to read and write, and easy for machines to parse and generate.

Key Features

  • Lightweight: Less verbose than XML, making it faster to transfer.
  • Language Independent: While derived from JavaScript, JSON is supported by almost every modern programming language (Python, Java, C#, etc.).
  • Self-describing: It's easy to understand what the data represents just by looking at it.

JSON Syntax Rules

JSON syntax is a subset of JavaScript object notation syntax:

  • Data is in name/value pairs.
  • Data is separated by commas.
  • Curly braces {} hold objects.
  • Square brackets [] hold arrays.

Example

{
  "users": [
    {
      "firstName": "John",
      "lastName": "Doe",
      "age": 30
    },
    {
      "firstName": "Jane",
      "lastName": "Smith",
      "age": 25
    }
  ]
}
